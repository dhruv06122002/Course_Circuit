HTML Image tags

The HTML <img> tag is used to embed an image in a webpage .
Images are not technically inserted into a webpage ,
they are linked with webpage .

The <img> tag has two required attributes :

  • src : It specifies the path of path of Image.
  • alt : It specifies an alternate text to Image.

SYNTAX

< img src = " url " alt = " alternatetext " >

Property
tag use Example
height Use to set height of Image < img src = " url " height="100" >
width Use to set width of Image < img src = " url " width="100" >

HTML Audio tags

The HTML < audio > tag is used to embed an Audio in a webpage .

The < audio > tag has two required attributes :

  • src : It specifies the path of path of Video.
  • control : Use to provide controls like play/pause.
  • source : Specifies alternate video file

SYNTAX

< audio controls >
< source src="url" type="mediatype" > < /audio >

Property
tag use Example
height Use to set height of Image < audio " height="100" >
width Use to set width of Image < audio width="100" >
type Use to specify Video media type < audio type="audio/mp3" >
 
<html>
<head>
<title>CODERA</title>
</head>
<body>
< audio src = " Audio.mp3 " type="audio/mp3" controls >
< /audio
</body>
</html>
