HTML List tags

The List tags are of three types:

1: Unordered list
2: Ordered list
3: Description list
 

Unordered List
The <ul> tag is used for creating an unordered list.
 

Attribute Description
type Specifies the style of the bullet.
Value for this attributes is Disc, circle, square.
Type Shape Example
Disc Colored circle
Circle Circle
Square Colored square

Ordered List
The <ol>tag is used for creating an ordered list.
 

Attribute Value Description
reversed Reversed Specifies the order of the list (reverse).
Start Number Specifies the initial number to start the list.
type A,a,i,I,1 Specifies the style of the list.
Type Shape Example
A Capital Alphabets
a Small Alphabets
I Roman numbers
i Smaller Roman numbers
1 Numeric value

Description List
The <dl> tag is used for creating an Description list.
 

Attribute Description
dt Use to define term or name
dd Use to describe each term or name

EXAMPLE

<html>
<head><title>Title of your web page</title></head>
<body>
<h3>UNORDERED LIST</h3>
<ul type="disc"><li>HTML</li></ul>
<ul type="circle"><li>CSS</li></ul>
<ul type="square"><li>PHP</li></ul>
<h3>ORDERED LIST</h3>
<ol type="A"><li>JAVASCRIPT</li></ol>
<ol type="a"><li>PYTHON</li></ol>
<ol type="i"><li>SQL</li></ol>
<ol type="I"><li>MYSQL</li></ol>
<ol type="1"><li>KOTLIN</li></ol>
<h3>DESCRIPTION LIST</h3>
<dl><dt>HTML</dt><dd>hypertext markup language</dd></dl>
</body>
</html>

OUTPUT

 

Title of your web page

UNORDERED LIST

  • HTML
  • CSS
  • PHP

ORDERED LIST

  1. JAVASCRIPT
  1. PYTHON
  1. SQL
  1. MYSQL
  1. KOTLIN

DESCRIPTION LIST

HTML
hypertext markup language
