HTML Form elements

Tag Description
<form> Defines an HTML form for user input.
<input> Defines an input controls.
<textarea> Defines a multiline input controls.
<label> Defines a label for input element.
<fieldset> Groups related elements in a form.
<legend> Defines a caption for a fieldset element.
<select> Defines a down drop list.
<optgroup> Defines a group of related options in down drop list.
<option> Defines an option in down drop list.
<button> Defines a clickable button/
<datalist> Specifies a list of pre-defines options for input controls.
<output> Defines the result of calculation.
