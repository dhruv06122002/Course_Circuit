HTML Attributes

Attribute Description Syntax
href Use to link pages with each other. <a href="filename.html">
src Use to link mediafiles with each other. <img src="mediafilename.format">
height Use to set height to element. <img src="mediafilename.format" height="size px" >
width Use to set width to element <img src="mediafilename.format" width="size px" >
alt Use to provide alternate link. <img src="mediafilename.format" alt="name" >
lang Use for declaring language. < lang="type">
title Use to provide extra information of element. < element title="text" >
