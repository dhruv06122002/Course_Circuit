HTML Formatting

Attribute Description Syntax Example
b Use to define bold text. <b> text </b> This is bold tag.
strong Use to define text with strong importance <strong> text </strong> This is strong tag.
i Use to define a part of voice or mood. <i> text </i> Welcome to HTML formatting.
em Use to define emphasized text. <em > text </em > This is emphasized tag
mark Use to highlight text <mark> text </mark> This is mark text.
small Use to define smaller tags <small> text </small> This is small text.
del Use to delete a text <del> text </del> This is del text.
ins Use to defines a text that has been initilized <ins> text </ins> This is ins text.
sub Use to define chemical formulas <sub> text </sub> This is sub text.
sup Use to define superscript text <sup> text </sup> This is sup text.
