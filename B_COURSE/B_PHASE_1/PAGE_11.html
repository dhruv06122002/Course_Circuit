HTML iframe , Filepath and Computer codes

HTML iframe

HTML <iframe> is used to display webpage within a webpage.

 

Syntax:
<iframe src="url" title="description">

 

Attributes Description
height Use to declare height.
width Use to declare width.
src Use to give location.
title Use to provide a title.

 

HTML Filepath

A filepath describes the location of a file in a website's folder structure.
Filepath are used whenlinking to external files like
: Webpages
: Mediafiles
: Stylesheet
: Javascripts

Path Description
src="picture.jpg" The picture.jpg is in same folder with current page.
src="images/picture.jpg" The picture.jpg is in image folder in current folder.
src="/images/picture.jpg" The picture.jpg is in image folder at the root of current web.
src="../picture.jpg" The picture.jpg is located in the folder one level up from the current folder.
 

HTML Computercodes

HTML contains several elements for defining user inuts and computer commands/codes

Tag Name Description Example
<code> Code tag Used to display content/code this is code tag.
<kbd> Keyboard input Used to define Keyboard inputs. this is kbd tag.
<samp> Sample code Used to define sample output this is samp tag.
<var> Variable tag Used to define variable in program. this is var tag.
