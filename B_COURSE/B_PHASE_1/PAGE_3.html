Tag Name Detail
<!DOCTYPE> DOCTYPE Used to Define documentation type.
<head> HEAD Used to declare heading section.
<title> TITLE Use to declare a title.
<pre> PRE Use to defines preformatted text.
<!--txt--> COMMENTS Use to Comment something.
<style> STYLE Use to provide internal CSS.
<script> SCRIPT Use to provide internal javascript.
 

EXAMPLE

<!doctype html> <!-- DOCTYPE declaration -->
<html> <!-- Head decleration -->
<head>
<title>Untitled Document</title> <!-- Title declaration -->
</head>
<style> </style> <!-- Style decleration -->
<body>
<h1>Hello!</h1>
<h2>Hello!</h2>
<h3>Hello!</h3>
<h4>Hello!</h4>
<h5>Hello!</h5>
<h6>Hello!</h6>
</body>
<script> </script> <!-- script decleration -->
</html>
