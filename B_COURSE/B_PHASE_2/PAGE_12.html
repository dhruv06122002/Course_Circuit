CSS Website Layout

Website Layout

A website is often divided into headers, menus, content and a footer:



Header

  • The <header> element specifies a header for a document or section.
  • The <header> element should be used as a container for introductory content.
  • You can have several <header> elements in one document.

    • Navigation Bar

  • The <nav> element defines a set of navigation links.
  • The <nav> element is intended for large blocks of navigation links.
  • However, not all links in a document should be inside a <nav> element.

    • Artical

  • The <article> element specifies independent, self-contained content.
  • It should be possible to read it independently from the rest of the website.
  • Examples of where an <article> element can be used: Forum post, Blog post, Newspaper article

    • Section

  • The <section> element defines a section in a document.
  • A section is a grouping of content, typically with a heading.
  • A Web site's home page could be split into sections for introduction, content and contact information.



    • Aside

  • The <aside> element defines some content aside from the content it is placed in (like a sidebar).
  • The aside content should be related to the surrounding content.



    • Footer

  • The <footer> element specifies footer for a document or section.
  • Footer typically contains the author of the document, copyright information, links to terms of use, contact information, etc.
  • You can have several <footer> elements in one document.

    • Sample Example


    EXAMPLE


    <!DOCTYPE html>
    <html>
    <body>
    <header>
    <h1>This is page heading (header)</h1>
    </header>
    <nav>
    <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">About Us</a></li>
    <li><a href="#">Contact Us</a></li>
    </ul>
    </nav>
    <article>
    <h1>This is article heading</h1>
    <p>Hello world! Hello world! Hello world!</p>
    </article>
    <aside>
    <img src="images/layout_2.jpg">
    </aside>
    <section>
    <h1>This is a section heading</h1>
    <p>Hello world! Hello world! Hello world! </p>
    </section>
    <footer><hr>Copyright (C) 2021. All rights reserved. </footer>
    </body>
    </html>
