CSS Shadow Effects

With CSS you can add shadow to text and to elements.
In these chapters you will learn about the following properties:

  • Text-shadow
  • Box-shadow
 

CSS Text Shadow


The CSS text-shadow property applies shadow to text.
In its simplest use, you only specify the horizontal shadow (2px) and the vertical shadow (2px):

 

Text shadow effect!


<!DOCTYPE html>
<html>
<head>
<style>
css1 {
text-shadow: 2px 2px;
}
css2 {
text-shadow: 2px 2px red;
}
css3 {
text-shadow: 2px 2px 5px red;
}
css4 {
color: white;
text-shadow: 2px 2px 4px #000000;
}
css5 {
text-shadow: 0 0 3px #FF0000;
}
</style>
</head>
<body>
<h1><css1>Text-shadow effect!</css1></h1>
<h1><css2>Text-shadow color effect!</css2></h1>
<h1><css3>Text-shadow blur effect!</css3></h1>
<h1><css4>Text-shadow white text with black shadow effect!</css4></h1>
<h1><css5>Text-shadow red neon glow shadow effect!</css5></h1>
</body>
</html>

Multiple Shadows


To add more than one shadow to the text, you can add a comma-separated list of shadows.
The following example shows a red and blue neon glow shadow:

 

Text shadow effect!


<!doctype html>
<html>
<head>
<style>
css6 {
text-shadow: 0 0 3px #FF0000, 0 0 5px #0000FF;
}
css7 {
color: white;
text-shadow: 1px 1px 2px black, 0 0 25px blue, 0 0 5px darkblue;
}
css8 {
color: yellow;
text-shadow: -1px 0 black, 0 1px black, 1px 0 black, 0 -1px black;
}
</style>
</head>
<body>
<h1><css6>Text-shadow effect with more than one shadow !</css6></h1>
<h1><css7>Text-shadow with white text with black, blue, and darkblue shadow effect!</css7></h1>
<h1><css8>Text-shadow a plain border around some text (without shadows) effect!</css8></h1>
</body>
</html>
