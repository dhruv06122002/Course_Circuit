CSS Shadow Effects

CSS Box Shadow


The CSS box-shadow property applies shadow to elements.
In its simplest use, you only specify the horizontal shadow and the vertical shadow:

 

Text shadow effect!


<!DOCTYPE html>
<html>
<head>
<style>
.cssbox1 {
width: 300px;
height: 100px;
padding: 15px;
background-color: yellow;
box-shadow: 10px 10px;
}
.cssbox2 {
width: 300px;
height: 100px;
padding: 15px;
background-color: yellow;
box-shadow: 10px 10px grey;
}
.cssbox3 {
width: 300px;
height: 100px;
padding: 15px;
background-color: yellow;
box-shadow: 10px 10px 5px red;
}
</style>
</head>
<body>

<h1>The box-shadow Property</h1>
<h1> </h1>
<div class="cssbox1" >This is a yellow element with a black box-shadow</div>
<h1> </h1>
<div class="cssbox2">This is a yellow element with a grey box-shadow</div>
<h1> </h1>
<div class="cssbox3">This is a yellow element with a blurred, red box-shadow</div>

</body>
</html>

Text shadow effect!
