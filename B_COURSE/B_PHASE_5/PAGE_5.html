PHP + MYSQL

Integration of PHP and MYSQL

It is possible to execute various commands of MYSQL.PHP provides various built-in functions which allows you to use MYSQL commands from PHP page . Thus you can integrate PHP with MYSQL.

What you should already have ?

  • Downloaded and installed a latest version of MySQL.
  • Created database
  • If you have not created a database then you would need root user and its password to create a database.
