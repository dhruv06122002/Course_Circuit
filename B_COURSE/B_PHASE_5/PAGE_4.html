Welcome to Phase 5 : Database(MYSQL)

Table and functions

Create Table

The SQL CREATE TABLE Statement
The CREATE TABLE statement is used to create a new table in a database.

SYNTAX:
CREATE TABLE table_name (
column1 datatype,
column2 datatype,
column3 datatype,
....
);
 

Insert data into Table

The INSERT INTO Statement
The INSERT INTO statement is used to INSERT data to a table in a database.

SYNTAX:
INSERT INTO table_name (column1, column2, column3, ...)
VALUES (value1, value2, value3, ...);
 

Update data into Table

The UPDATE Statement
The UPDATE statement is used to UPDATE data in a table in a database.

SYNTAX:
UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition;
 

Delete data in Table

The DELETE Statement
The DELETE statement is used to DELETE existing data in a table in a database.

SYNTAX:
DELETE FROM table_name WHERE condition;
 

Delete Table

The SQL DROP TABLE Statement
The DROP TABLE statement is used to DELETE a table in a database.

SYNTAX:
DROP TABLE table_name;
